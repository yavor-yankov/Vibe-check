## Quick Start

**T-14 days:** Read `LAUNCH_PLAN.md` → complete pre-launch checklist
**T-7 days:** Set up Resend sequences from `EMAIL_SEQUENCES.md` and schedule social from `AUTOMATION_GUIDE.md`
**T-3 days:** Submit to directories listed in `LAUNCH_PLAN.md`
**Launch day:** Follow hourly sequence in `LAUNCH_PLAN.md`, use copy from `CHANNEL_COPY.md`
